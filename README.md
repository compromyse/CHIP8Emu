# CHIP8Emu
An emulator for CHIP-8 written in C++

![image](https://github.com/compromyse/CHIP8Emu/assets/71056504/f87e9e73-f880-4d60-a61b-cdb40791211b)

## Building

### Requirements

* CMake
* SDL2
* Make

### Running the build

```sh
mkdir build; cd build
cmake ..
make
```

## Syntax
```sh
./chip8emu <Scale> <Delay> <ROM>
```
